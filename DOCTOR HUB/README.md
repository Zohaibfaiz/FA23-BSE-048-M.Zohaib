# ✨ Key Features

## 👨‍⚕️ Patients

* Browse Doctors
* Search Specialists
* Book Appointments
* Manage Medical History
* Upload Payment Proofs
* View Prescriptions
* Manage Profile

## 🩺 Doctors

* Appointment Management
* Patient Records
* Prescription Writing
* Clinic Scheduling
* Profile Management

## 👨‍💼 Administrators

* User Management
* Doctor Approval
* Reports & Analytics
* Payment Verification
* Audit Logs
* System Configuration

## 👩‍💻 Assistants

* Queue Management
* Patient Check-in
* Appointment Handling
* Payment Verification

* React 19
* Vite
* Tailwind CSS
* React Router v7
* Axios
* Framer Motion
* React Hook Form
* Zod

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Nodemailer
* Multer
* Helmet
* CORS
* Express Rate Limit

# 🚀 Quick Start

## Clone Repository

```bash
git clone https://github.com/your-username/doctor-hub.git

cd doctor-hub
```

## One Click Setup

```bash
install-and-run.bat
```

### Application URLs

Frontend:
http://localhost:5173

Backend:
http://localhost:5000/api

Health Check:
http://localhost:5000/api/health
