# 📌 Overview

**SecureVote Pro** is a modern and secure online election management platform designed to provide transparent, anonymous, and real-time digital voting.

The platform allows organizations, universities, and institutions to conduct elections securely with modern authentication, encrypted voting, real-time analytics, and enterprise-level administration features.

The system supports three major user roles:

- 👤 Voters
- 🗳️ Election Creators
- 🛡️ Admins

Built using modern technologies like React, Vite, Tailwind CSS, and Supabase, the platform focuses heavily on:

- Data Privacy
- Anonymous Voting
- Real-Time Monitoring
- Enterprise Security
- Responsive User Experience

# ⚡ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone <repository-url>
```

---

## 2️⃣ Install Dependencies

```bash
npm install
```

---

## 3️⃣ Configure Environment Variables

Create a `.env` file and add:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

---

## 4️⃣ Start Development Server

```bash
npm run dev
```
